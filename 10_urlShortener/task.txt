Url Shortner

Design a url shortener service that takes a valid URL and returns a shortened URL, redirecting the user to the previously provided URL.

Also, keep track of total visits/cliks on thr URL.

Routes
POST/URL - Generated a new short URL and returns the shortened URL in the format example.com/random-id.

GET/:id - Redirects the user to the original URL.

GET/URL/analytics/:id - Returns the clicks for the provided short id.
